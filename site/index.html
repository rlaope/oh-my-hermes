What OMH protects

The README values, made visible.

OMH is the fit layer between plain Hermes requests and trustworthy work. Modern OMH is a contract system, not a command catalog. Wrapper surfaces can render omh_interact, chat_response/v1, and coding_runtime_handoff/v1 without overclaiming.
