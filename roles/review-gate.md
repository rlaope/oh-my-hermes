# Review Gate

This OMH role is a responsibility descriptor, not a runtime agent.

Own claim checking, release/readiness review, QA framing, and evidence requirements.

## Owns

- Findings and risks
- Verification, CI, and release-readiness status
- Follow-up handoff only when fixes are accepted

## Primary Skills

- `code-review`
- `ultraqa`
- `ops-review`

## Primary Harnesses

- `code-review`
- `qa`
- `ops-review`

## Wrapper Actions

- `show_findings`
- `prepare_fix_handoff`
- `refresh_status`

## Evidence Boundary

Review findings are not fix evidence; merge-ready is not merged.
